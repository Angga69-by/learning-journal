# Jurnal Minggu 7: Pengenalan Tools Disassembler Ghidra & Debugger x64dbg

## Apa yang Dipelajari?
Minggu ini saya belajar mengoperasikan dua jenis alat bantu (*tools*) utama yang menjadi senjata wajib seorang *Reverse Engineer*:
1. **Disassembler / Decompiler (Ghidra):** Software open-source buatan NSA yang mampu membaca kode mesin, mengubahnya menjadi assembly, dan melakukan rekonstruksi balik menjadi bahasa C semu (*Pseudo-code*). Ini sangat membantu dalam memahami logika algoritma internal aplikasi tanpa source code asli.
2. **Debugger (x64dbg):** Alat untuk analisis dinamis (*dynamic analysis*). x64dbg memungkinkan saya mengeksekusi program baris demi baris, memantau perubahan nilai register real-time, melihat isi stack, serta memasang *breakpoint* pada alamat instruksi tertentu.

## Kendala dan Solusi
* **Kendala:** Antarmuka (*interface*) Ghidra dan susunan windows di x64dbg terlihat sangat kompleks dan membingungkan bagi pemula.
* **Solusi:** Mengikuti tutorial dasar terpandu, fokus membaca jendela dekompilasi utama, dan menghafal tombol pintas penting (seperti F7 untuk step-into dan F8 untuk step-over).
